# LocationSimulator
 iOS 設備的 GPS 定位模擬器
 # 使用手冊

iOS 位置模擬工具，專為需要靈活控制地理位置的開發者與測試人員設計。本版本引入了全新的智慧環境識別、路徑管理以及 POI 收集系統。

## 🛠️ 核心功能介紹

### 1. 位置控制 (Location Control)
透過主面板，您可以進行精確的位置跳轉：
*   **座標搜尋**：支援直接輸入經緯度或地址關鍵字進行搜尋。
*   **實時地名解析**：搜尋後會顯示當地的詳細行政區域資訊。
*   **虛擬搖桿**：在地圖下方提供直覺的圓形搖桿介面，支援細微的位移操作。
*   **開始與停止**：一鍵開啟模擬（Simulate）或恢復真實定位（Stop）。

### 2. 環境資訊 (Environment)
側邊欄頂部提供當前環境的動態資訊：
*   **深色模式**：支援一鍵切換 Dark Mode 視覺主題。
*   **虛擬當地時鐘**：根據您模擬的地點，自動顯示當地的實時日期與時間。
*   **地理位置標籤**：同步顯示模擬點所屬的國家與城市，方便跨境測試。
*   **穩定性警示**：若時區解析出現誤差，會出現 ⚠ 圖示提示當前為估計時間。

### 3. 設備連接管理 (Device Connection)
支援多種連線方式，簡化配對流程：
*   **無線連線 (WiFi)**：透過自動掃描功能識別區域網路內的 iOS 設備。
*   **連線狀態**：清單中會顯示設備名稱及其當前連線協定（如 WiFi）。
*   **一鍵刷新**：點擊右上角刷新圖示重新掃描可用設備。

### 4. 座標收藏與 Big Flowers (POI)
針對特定遊戲或應用場景設計的點位管理：
*   **Saved Locations**：收藏您最常去的座標點，並支援匯入/匯出功能。
*   **Big Flowers 管理**：特別針對特定場景設計，具備 40 公尺範圍顯示與自動行政區分類功能，支援批次隱藏與管理。

## 🚀 快速上手教學

### 第一步：連接設備
1. 確保您的 iOS 設備與電腦在同一網路環境，或透過 USB 連接。
2. 在 **Device Connection** 面板點擊設備旁的插頭圖示進行配對。

### 第二步：設定座標
1. 您可以透過 **Location Control** 搜尋座標，或直接在地圖上**雙擊滑鼠**來放置藍色定位點。
2. 檢查側邊欄 **Environment**，確認當地時間與區域資訊是否正確。

### 第三步：開始模擬
1. 點擊 **Simulate** 按鈕，您的 iOS 設備定位將立即同步至目標位置。
2. 若需微調位置，可使用面板中央的**搖桿**進行操作。

### 第四步：路徑與工具欄操作
地圖側邊提供豐富的工具列（參考圖片 1）：
*   **雲端上傳/下載**：批次匯入與導出 GPX 檔案或路徑資料包。
*   **🟢 綠色播放按鈕 (Drawing Playback)**：用於播放您目前正在地圖上「即時繪製」的綠色暫存路徑。這適合在正式儲存前先進行測試。
*   **🔵 藍色播放按鈕 (Saved Playback)**：用於播放您從側邊欄選中的「已儲存路徑」（地圖上顯示為虛線藍色）。
*   **🌸 Big Flower 放置模式**：點擊花朵圖示進入該模式，在地圖上點擊即可放置具備 40 公尺範圍圈的興趣點。這些點位會自動按行政區分類，方便在側邊欄進行批次管理。
*   **繪製模式**：切換手動路徑繪製（鉛筆圖示）或自動吸附路徑。
*   **速度調整**：點擊底部速度標籤（如 6 km/h）切換不同的移動模式（步行、騎車、開車）。

## 📦 安裝與測試方法
1. 下載 `LocationSimulator.exe` 。
2. **按滑鼠右鍵點擊 `LocationSimulator.exe`，選擇「以系統管理員身分執行 (Run as Administrator)」**。
3. 啟動後畫面的黑色視窗會進行自動配置，隨後會自動開啟瀏覽器進入地圖操作介面。
